# Changelog

## [3.0.0] — 2026-06

### Перший публічний реліз (Release 3.0)

Публічна версія інструментарію МіО іШГБ, заснованого на методиці DIY4Change
(Oleksii Kovalenko, PhD) та практиці впровадження CSDF NGO.

### Додано
- Генерація аналітичного звіту МіО іШГБ для 1–N громад
- 6 індикаторів із перевіреними формулами (Інд. 2, 4, 6 — уточнені знаменники)
- 4 діаграми: bar chart (Інд. 1/2), воронка (Інд. 3), кластерний bar (Інд. 4/5), bubble (Інд. 6)
- Розділ 3.3 «Інклюзивна участь дітей з ООП та інвалідністю» (якщо дані надані)
- Підтримка 1–N громад у таблицях і діаграмах
- Гнучка титульна сторінка — параметри задає користувач (організація, рік, підготовлено)
- LibreOffice re-save для сумісності з MS Word на Windows
- Майстер-промпт v1 для запуску через Claude

### Примітка щодо версіонування
Версія 3.0 відображає загальну лінію розвитку інструментарію МіО іШГБ
в рамках методики DIY4Change. Попередні версії (1.0, 2.0) були внутрішніми
і не публікувались відкрито.
