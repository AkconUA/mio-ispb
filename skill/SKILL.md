# іШГБ: Аналітичний звіт МіО — повний робочий процес (v3.0)

## Що робить цей скіл

Отримує Excel-файл із моніторинговою таблицею іШГБ → розраховує 6 індикаторів
по кожній громаді → порівнює з еталонними значеннями → генерує:
- **Аналітичний_звіт_іШГБ_[рік].docx** — єдиний повний звіт,
  де в розділі 2.3 вбудовані панель індикаторів та всі 4 PNG-діаграми,
  а розділ 3.3 містить таблицю залучення дітей з ООП та інвалідністю (якщо дані надані).

> Окремий файл інфографіки НЕ генерується — усі діаграми інтегровано в звіт.

## КРОК 1. Розрахунок 6 індикаторів

```
1. Рівень залучення (%)    = (автори + виборці) / потенційні × 100
2. Охоплення шкіл (%)      = шкіл_учасниць / шкіл_всього × 100   ← ЗАКЛАДИ, не учні
3. Частка авторів (%)      = автори / навчені × 100
4. Проєктів на 100 авторів = проєктів_на_голосуванні / авторів × 100  ← АВТОРИ, не потенційні
5. Якість (%)              = проєктів_на_голосуванні / розроблених × 100
6. Активізація             = виборці / авторів → «1 : X»  ← АВТОРИ, не навчені
```

> ⚠️ **Критичні уточнення:**
> - Інд. 2: знаменник = **шкіл_всього** (заклади освіти, не учні).
> - Інд. 4: знаменник = **автори** (не потенційні учні).
> - Інд. 6: знаменник = **автори** (не навчені учні).

**Зведені показники** = сума значень усіх громад, що подали звіт (НЕ середнє).

### Еталонні значення

| Індикатор | Еталон |
|---|---|
| 1. Рівень залучення | 72% |
| 2. Охоплення шкіл | 50% |
| 3. Частка авторів | 26,2% |
| 4. Проєктів на 100 авторів | 10,24% |
| 5. Якість | 84,39% |
| 6. Активізація | 1 : 3 |

Якщо користувач надає інші еталонні значення — використовуй їх.

## КРОК 3. Генерація звіту (.docx)

```javascript
function image(filename, w, h) {
  return new Paragraph({
    alignment: AlignmentType.CENTER,
    children: [new ImageRun({
      type: 'png', data: fs.readFileSync(filename),
      transformation: { width: w, height: h },
      altText: { title: filename, description: filename, name: filename }
    })]
  });
}
// Розміри для A4 (поля 2–2.5 см):
image('/home/claude/charts/chart1_donut.png', 570, 260)
image('/home/claude/charts/chart2_funnel.png', 570, 280)
image('/home/claude/charts/chart3_bars.png',  570, 250)
```


### Титульна сторінка — параметри (запитати у користувача)


## Типові помилки та рішення

| Помилка | Рішення |
|---|---|
| `ax.pie()` з `width` | тільки через `wedgeprops=dict(width=…)` |
| `PageNumber` не конструктор | `new TextRun({ children: [PageNumber.CURRENT] })` |
| Файл не відкривається у Word | LibreOffice re-save (Крок 4) |
| `WidthType.PERCENTAGE` | завжди `WidthType.DXA` |
| Булети unicode | тільки `LevelFormat.BULLET` через numbering config |
| Кирилиця → LibreOffice помилка | конвертувати через тимчасове ASCII-ім'я |
| `ImageRun` без `altText` | завжди `altText: { title, description, name }` |
| Воронка — KeyError `total_students` | використовувати `potential` як max_val |
| Інд. 2 від учнів | знаменник = **шкіл_всього** |
| Інд. 4 від потенційних | знаменник = **авторів** |
| Інд. 6 від навчених | знаменник = **авторів** |
| Нова громада не в зведенні файлу | рахувати вручну, не довіряти зведеному рядку xlsx |
