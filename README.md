# іШГБ МіО — Аналітичний звіт: збірка інструментів

**Інклюзивний Шкільний Громадський Бюджет (іШГБ)**  
Автоматизована система генерації аналітичного звіту моніторингу та оцінки (МіО)

---

## 📦 Що є в цьому репозиторії

```
ishgb-mio/

│   └── MiO_iSPB_template.xlsx        ← Шаблон моніторингової таблиці іШГБ
│

│   ├── ishgb_auto.py                 ← Допоміжний скрипт автоматизації
│   ├── generate_report.js            ← Генератор Word-звіту (Node.js / docx)
│   ├── make_charts.py                ← Генератор діаграм (Python / matplotlib)
│   └── fix_funnel.py                 ← Скрипт виправлення воронкової діаграми
│

    └── SKILL.md                      ← Технічна карта для Claude (скіл)
```

## 🚀 Як використовувати (швидкий старт)

### Для нетехнічних користувачів
1. Завантажте `docs/Інструкція_іШГБ.docx` і прочитайте її
2. Заповніть `templates/MiO_iSPB_template.xlsx` даними вашої громади
3. Відкрийте `docs/Мастер-промпт_іШГБ.docx`, скопіюйте промпт
4. Перейдіть на [claude.ai](https://claude.ai), завантажте Excel, вставте промпт

### Для автоматизованої генерації (через Claude)
1. Надайте Claude файл `skill/SKILL.md` і Excel з даними
2. Напишіть: *«Зроби звіт іШГБ. Проєкт: [назва]. Громади: [перелік]. Автор: [ПІБ]»*
3. Claude згенерує два файли: звіт + інфографіку

## 📊 Структура звіту (що генерується)

| Розділ | Зміст |
|--------|-------|
| Титульна сторінка | Організація, назва, автор, рік |
| 0. Вступ | Контекст проєкту, джерела даних |
| 1. Саммері МіО | Методологія + 5 ключових тез |
| 2. Результати | Матриця 6 індикаторів + детальний аналіз + інфографіка |
| 3. Висновки | Оцінка vs еталон + рекомендації для ОМС/НУО/педагогів/донорів |

## 📐 6 Індикаторів іШГБ та еталонні значення

| # | Індикатор | Формула | Еталон |
|---|-----------|---------|--------|
| 1 | Рівень залучення | (автори + виборці) / потенційні × 100 | **72%** |
| 2 | Охоплення | потенційні / всього учнів × 100 | **50%** |
| 3 | Частка авторів | автори / навчені × 100 | **26,2%** |
| 4 | Проєктів на 100 учнів | розроблені / потенційні × 100 | **10,24** |
| 5 | Якість проєктів | на голосування / розроблені × 100 | **84,39%** |
| 6 | Активізація | виборці / навчені → «1 : X» | **1 : 3** |

## 🎨 Діаграми (окремий файл)

| Рис. | Тип | Індикатор |
|------|-----|-----------|

## 📋 Шаблон моніторингової таблиці

Файл `templates/MiO_iSPB_template.xlsx` містить стандартну зведену таблицю індикаторів іШГБ з колонками:

- К-ть шкіл (всього / взяло участь)
- SPB12: к-ть навчених учнів
- К-ть авторів проєктів
- К-ть розроблених проєктів
- К-ть проєктів, відхилених / відхилених кошторисів
- К-ть презентованих / вийшли на голосування
- К-ть школярів, що проголосували
- К-ть переможців, реалізованих, суми
- SPB13: всього учнів / потенційно можуть брати участь
