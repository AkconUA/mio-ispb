**Інклюзивний Шкільний Громадський Бюджет (іШГБ / iSPB)**

---

## 🚀 Як користуватись (коротко)

1. Відкрий `docs/Майстер-промпт_iSPB_v2.txt`, заміни `< >` на дані свого проєкту
2. Зайди на [claude.ai](https://claude.ai), завантаж Excel-файл моніторингу
3. Вкинь промпт і надішли
4. Перевір розраховані індикатори → напиши **«Підтверджую, продовжуй»**
5. Завантаж готовий `.docx` файл

Детально — у `docs/Інструкція_iSPB_v2.docx`

## 📊 6 індикаторів іШГБ — формули v2.0

| # | Індикатор | Формула |
|---|---|---|
| 1 | Рівень залучення (%) | (автори + виборці) / потенційні × 100 |
| 2 | **Охоплення шкіл (%)** | **шкіл-учасниць / шкіл всього × 100** |
| 3 | Частка авторів (%) | автори / навчені × 100 |
| 4 | **Проєктів на 100 авторів (%)** | **проголосованих / авторів × 100** |
| 5 | Якість проєктів (%) | проголосованих / розроблених × 100 |
| 6 | Активізація | виборці / навчені → «1 : X» |

> ⚠️ **Увага:** формули 2 і 4 виправлені у v2.0 — перевір якщо рахував за v1.

### Еталонні значення

| Індикатор | Еталон |
|---|---|
| Рівень залучення | 72% |
| Охоплення шкіл | 50% |
| Частка авторів | 26,2% |
| Проєктів на 100 авторів | 10,24% |
| Якість проєктів | 84,39% |
| Активізація | 1 : 3 |

## 📄 На виході — один файл

`Аналітичний_звіт_іШГБ_[рік].docx` — 13+ сторінок:
- Титул, зміст, вступ із таблицею реквізитів
- Таблиця 6 формул + 5–6 ключових тез
- Порівняльна матриця з кольоровим маркуванням ▲/▼
- Детальний аналіз по кожній громаді (формула + числа + інтерпретація)
- **Інфографіка вбудована в розділ 2.3** (панель + 4 діаграми)
- Висновки та рекомендації (ОМС / координатори / педагоги / донори)

## 🔄 Що нового у версії 2.0

## 🛠 Технічний стек

- **Claude AI** (claude.ai) — генерація тексту та `.docx`
- **Node.js + docx npm** — формування Word-документу
- **Python 3 + matplotlib** — генерація PNG-діаграм (`scripts/generate_charts_iSPB.py`)
- **LibreOffice** (headless) — фінальна конвертація для сумісності з MS Word
